‚ûï 3. Managing Transactions

Adding a New Transaction

Step 1: Navigate to Add Transaction

Click "Add Transaction" in the sidebar menu

Step 2: Choose Transaction Type

Select either:

  • Income - Money received (consultations, lab tests, etc.)
  • Expense - Money spent (supplies, salaries, utilities, etc.)

Step 3: Fill in Transaction Details

  • Amount: Enter the transaction amount
  • Currency: Choose SSP or USD
  • Department: Select the relevant department
  • Description: Brief description of the transaction
  • Date: Transaction date (defaults to today)

Step 4: Add Insurance (for Income)

If this is insurance-related income:

  • Check "Insurance Transaction"
  • Select the insurance provider
  • Enter policy or claim number if available

Step 5: Upload Receipt (Optional)

Drag and drop or click to upload receipt images for record keeping

Step 6: Save Transaction

Click "Add Transaction" to save

‚ö†Ô∏è Important: Always double-check the amount and currency before saving. Transactions can be edited later, but it's best to enter them correctly the first time.

Department Categories

  • Consultation - Doctor visits and consultations
  • Laboratory - Lab tests and diagnostics
  • Ultrasound - Ultrasound examinations
  • X-Ray - X-ray services
  • Pharmacy - Medicine and pharmaceutical sales
  • Other - Miscellaneous income/expenses

üë• 5. Patient Volume Tracking

Recording Daily Patient Numbers

Step 1: Navigate to Patient Volume

üë§ 6. User Management (Admin Only)

‚ö†Ô∏è Admin Access Required: These features are only available to users with administrator privileges.

Managing Clinic Users

Adding New Staff

  1. Click "User Management" in the sidebar
  2. Click "Add New User" button
  3. Fill in staff details (name, email, role)
  4. Choose role: Admin or Staff
  5. Set initial password
  6. Save the new user

User Roles

  • Admin: Full access including user management and system settings
  • Staff: Can add transactions, view reports, and manage patient volume

Deactivating Users

To temporarily disable a user account:

  1. Find the user in the user list
  2. Click the three-dot menu
  3. Select "Deactivate"
üí° Security Tip: Regularly review user accounts and deactivate accounts for staff who no longer work at the clinic.

üîß 8. Troubleshooting

Common Issues & Solutions

Cannot Log In

  • Check username and password spelling
  • Ensure Caps Lock is off
  • Contact your administrator for password reset
  • Try refreshing the page

Page Not Loading

  • Check your internet connection
  • Refresh the browser page (F5 or Ctrl+R)
  • Clear browser cache and cookies
  • Try a different browser

Transaction Not Saving

  • Check all required fields are filled
  • Ensure amount is a valid number
  • Check internet connection
  • Try logging out and back in

Mobile Navigation Issues

  • Tap the hamburger menu (‚ò∞) to open sidebar
  • Tap outside the sidebar to close it
  • Refresh the page if menu is unresponsive

Getting Help

üìû Support Contact: If you encounter issues not covered in this guide, contact your system administrator or IT support team for assistance.

Best Practices

  • Daily Data Entry: Enter transactions daily for accurate reporting
  • Regular Backups: Monthly reports serve as data backups
  • Consistent Categories: Use the same departments for similar transactions
  • Receipt Storage: Upload receipts when possible for audit trails
  • Currency Accuracy: Double-check currency selection for each transaction
